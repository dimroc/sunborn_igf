SunBorn began in January of 2006 with the first beta released on November 10th 2006. The goal is to deliver a 3rd person multiplayer action with the aim of knocking your opponent off the 'islands'. Everything is written in C++ using DirectX, with networking built upon RakNet. Requirements are projected to be any computer that has a Shader Model 2.0 video card.

We are in need of a 24/7 uptime computer with a static IP address for our Master Server. This is the server that all player hosted servers let know they exist so clients can populate their server browser.
